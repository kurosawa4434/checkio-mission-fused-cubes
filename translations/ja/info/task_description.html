立方体を組み合わせたオブジェクトの体積を求めるミッションです。 あなたは立方体の詳細 (4つの整数のタプル : X座標, Y座標, Z座標, 辺の長さ) のリストを与えられます。

  • 各座標は最小の値です。
  • 全ての辺は座標軸に平行です。

部分を共有しているか面が接触している場合はそれらを一つのオブジェクトとみなします。 全てのオブジェクトの体積のリスト(またはiterble)を返してください。

Example:

入力:

  • 立方体の詳細(4つの整数のタプル : X座標, Y座標, Z座標, 辺の長さ) のリスト

    • 出力: 各オブジェクトの体積のリスト(またはiterble)
